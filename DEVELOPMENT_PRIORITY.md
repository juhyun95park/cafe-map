# 개발 우선순위 및 일정

## 핵심 기능 3개

PRD 분석 결과, 다음 3가지 핵심 기능을 식별했습니다:

### 1. 위치 기반 카페 목록 및 지도 표시
**설명**: 사용자의 현재 위치를 기반으로 주변 1km 반경 내 카페를 지도와 목록으로 표시
**완료 기준**: 
- 위치 기반으로 주변 카페를 지도와 목록으로 보여줄 수 있다
- 강남/홍대 지역의 카페 50개 데이터가 사전 탑재되어 있어야 한다

### 2. 실용 정보 필터링 시스템
**설명**: '조용함', '콘센트 많음', '주차 가능' 등 실용적인 조건으로 카페 필터링
**완료 기준**: 
- 조용함, 콘센트 등 최소 3개 이상의 핵심 필터가 정상적으로 작동한다

### 3. 태그 기반 리뷰 시스템
**설명**: 별점 대신 태그 기반 리뷰 작성 및 조회 (예: "콘센트 자리 많아요", "오래 앉아 있기 좋아요")
**완료 기준**: 
- 카페 상세 페이지에서 기본 정보와 태그 기반 리뷰를 확인할 수 있다
- 사용자가 신규 태그 리뷰를 남길 수 있다
- 강남/홍대 지역의 리뷰 100개 데이터가 사전 탑재되어 있어야 한다

## 개발 순서 및 예상 소요 시간

## 개발 시 주의사항

### 반드시 구현해야 할 것 (Definition of Done)
✅ 위치 기반으로 주변 카페를 지도와 목록으로 보여줄 수 있다
✅ 조용함, 콘센트 등 최소 3개 이상의 핵심 필터가 정상적으로 작동한다
✅ 카페 상세 페이지에서 기본 정보와 태그 기반 리뷰를 확인할 수 있다
✅ 사용자가 신규 태그 리뷰를 남길 수 있다
✅ 강남/홍대 지역의 카페 50개, 리뷰 100개 데이터가 사전 탑재되어 있어야 한다

### 절대 구현하지 말아야 할 것 (Nope)
❌ 회원가입 및 로그인 시스템
❌ 카페 예약 또는 결제 연동 기능
❌ 커뮤니티, 소셜 기능 (피드, 댓글, 팔로우 등)

### 기술적 제약사항
- 지도 로딩이 5초 이상 걸리거나 실패하는 경우 → 출시 불가
- 필터 적용 시 결과가 나오지 않거나 앱이 멈추는 현상 → 출시 불가
- 리뷰 작성이 DB에 저장되지 않는 문제 → 출시 불가
