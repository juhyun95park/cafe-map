# 카공여지도

서울 강남, 홍대 지역의 작업/공부하기 좋은 카페를 실용적인 정보로 찾아보는 모바일 웹 플랫폼입니다.

## 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Map Library**: react-leaflet + OpenStreetMap (API 키 불필요)

## 주요 기능

### Phase 1: 위치 기반 카페 목록 및 지도 표시 ✅
- 사용자 위치 기반 주변 1km 반경 카페 검색
- react-leaflet + OpenStreetMap을 통한 지도 표시 (API 키 불필요)
- 카페 목록 카드 형태로 표시
- 강남/홍대 지역 카페 50개 초기 데이터
- 사용자 위치 주변 동적 카페 생성

### Phase 2: 실용 정보 필터링 시스템 (예정)
- 조용함, 콘센트 많음, 주차 가능 등 필터 기능

### Phase 3: 태그 기반 리뷰 시스템 (예정)
- 태그 기반 리뷰 작성 및 조회
- 리뷰 100개 초기 데이터

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 프로젝트 구조

```
cafe-map/
├── app/                    # Next.js App Router
│   ├── page.tsx           # 홈 페이지
│   ├── cafe/[id]/         # 카페 상세 페이지
│   └── layout.tsx         # 루트 레이아웃
├── components/            # 컴포넌트
│   ├── cafe/             # 카페 관련 컴포넌트
│   └── ui/               # UI 컴포넌트
├── lib/                  # 유틸리티 및 데이터
│   ├── data/            # 초기 데이터
│   └── utils/           # 유틸리티 함수
└── types/               # TypeScript 타입 정의
```

## Definition of Done 체크리스트

### Phase 1 완료 기준
- ✅ 위치 기반으로 주변 카페를 지도와 목록으로 보여줄 수 있다
- ✅ 강남/홍대 지역의 카페 50개 데이터가 사전 탑재되어 있다

### Phase 2 완료 기준 (예정)
- ⏳ 조용함, 콘센트 등 최소 3개 이상의 핵심 필터가 정상적으로 작동한다

### Phase 3 완료 기준 (예정)
- ⏳ 카페 상세 페이지에서 기본 정보와 태그 기반 리뷰를 확인할 수 있다
- ⏳ 사용자가 신규 태그 리뷰를 남길 수 있다
- ⏳ 강남/홍대 지역의 리뷰 100개 데이터가 사전 탑재되어 있어야 한다
